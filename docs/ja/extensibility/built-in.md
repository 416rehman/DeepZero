## 組み込みプロセッサ

| 識別子 | タイプ | 機能 |
| ------------- | ---- | ------------- |
| `file_discovery` | Ingest | ファイルシステムの再帰的な走査と暗号化ハッシュ。 |
| `metadata_filter` | Map | ブール制約のチェックとフィールドの重複排除。 |
| `hash_exclude` | Map | 組み込みリストまたはフラットファイルを使用した暗号化ハッシュによる除外。 |
| `generic_llm` | Map | Jinja2 テンプレートのレンダリング、LLM の呼び出し、および応答の解析。 |
| `generic_command` | Map | コンテキスト変数の置換を伴う任意のシェルコマンドの実行。 |
| `top_k` | Reduce | 数値スカラー値に基づいてサンプルセットを切り捨てます。 |
| `sort` | Reduce | データセットを縮小せずにサンプルを決定論的に並べ替えます。 |
