大規模なゼロデイ脆弱性探索。

DeepZeroは自動化された脆弱性研究のための高並行オーケストレーションエンジンです。膨大なデータセットに対して、バイナリの解析、ヒューリスティクスの実行、およびカスタマイズされたマルチステージパイプラインを使用したLLMのスケーリングを行います。

はじめる ソースコード

大規模な並行委譲

Snappy Driver Installer のような巨大なコーパスを取り込みます。DeepZero はハードウェア上でネイティブに PE 解析、Ghidra ヘッドレスデコンパイル、静的解析を並列化します。

ヒューリスティックおよび AI 評価

LLM はパイプラインの一つのステージにすぎません。DeepZero は Windows IOCTL 表面をフィルタリングし、loldrivers.io ハッシュを除外し、Semgrep ルールを実行し、最も悪用の可能性が高い候補に対してのみ LLM 評価を使用します。

耐障害性のあるオーケストレーション

数週間にわたる調査向けに構築されています。DeepZero はすべてのサンプルのアトミックな状態を永続的に追跡します。実行をいつでも中断し、進行状況を失うことなく再開できます。
