## Создание Пользовательских Процессоров

Пользовательская логика интегрируется в DeepZero путем наследования типизированных абстракций в `deepzero.engine.stage`.

### Контекст Процессора (`ProcessorContext`)

Объект `ctx`, внедряемый в каждый хук жизненного цикла, обеспечивает системный контекст:
- `ctx.pipeline_dir`: Корневой каталог запущенного конвейера.
- `ctx.global_config`: TypedDict, содержащий `settings`, `knowledge` и `model`.
- `ctx.llm`: Экземпляр прокси для абстракции API поставщиков через LiteLLM.
- `ctx.log`: Настроенный логгер.

### Определения Конфигурации

Используйте `@dataclass` с именем `Config` для определения принимаемых конфигураций YAML. Движок анализирует [YAML Конвейера]({{ '/ru/reference/pipeline-yaml.html' | relative_url }}) и создает экземпляр вашего объекта `Config`.
