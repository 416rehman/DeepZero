## Встроенные процессоры

| Идентификатор | Тип | Функциональность |
| ------------- | --- | ---------------- |
| `file_discovery` | Ingest | Рекурсивный обход файловой системы и криптографическое хеширование. |
| `metadata_filter` | Map | Проверка логических ограничений и дедупликация полей. |
| `hash_exclude` | Map | Криптографическое исключение с помощью встроенных списков или плоских файлов. |
| `generic_llm` | Map | Рендеринг шаблонов Jinja2, вызов LLM и парсинг ответов. |
| `generic_command` | Map | Произвольное выполнение shell-команд с подстановкой контекстных переменных. |
| `top_k` | Reduce | Усечение набора образцов на основе числовых скалярных значений. |
| `sort` | Reduce | Детерминированное переупорядочивание образцов без сокращения набора данных. |
