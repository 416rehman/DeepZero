## Структура репозитория

```text
src/deepzero/
├── __init__.py          # Определения SemVer
├── __main__.py          # Точка входа модуля
├── cli.py               # Интерфейс приложения Click
├── api/                 # Конечные точки Starlette REST
├── engine/              # Основная логика выполнения и оркестрации
└── stages/              # Стандартная библиотека процессоров

processors/              # Предоставленные процессоры
pipelines/               # Объявления и конфигурации конвейеров
tests/                   # Тестовый пакет Pytest
```
