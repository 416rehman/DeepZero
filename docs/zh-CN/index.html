大规模发现零日漏洞。

DeepZero 是一个用于自动化漏洞研究的高并发编排引擎。通过定义自定义的多阶段流水线，解析二进制文件、运行启发式算法，并在海量数据集上大规模编排 LLM（大型语言模型）。

开始使用 查看源码

海量并行分流

摄取像 Snappy Driver Installer 这样的大型语料库。DeepZero 可以在你的硬件上原生并行执行 PE 解析、Ghidra 无头反编译和静态分析。

启发式和人工智能评估

LLM（大型语言模型）仅仅是流水线中的一个阶段。DeepZero 能够过滤 Windows IOCTL 接口面、排除已知的 loldrivers.io 哈希值、运行 Semgrep 规则，并只在最高信号的候选对象上使用 LLM 评估可利用性。

高弹性的调度编排

专为持续一周的漏洞捕获行动而设计。DeepZero 追踪每个样本的原子状态。你可以随时中断运行，并在之后瞬间恢复，且不会丢失任何进度。
